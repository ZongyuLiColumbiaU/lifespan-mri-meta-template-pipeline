## Suggested repo website text

This repository implements a hierarchical, age-aware ANTs workflow for lifespan structural MRI studies. It builds unbiased age-bin templates, constructs a second-stage meta-template, registers subjects into a shared meta-space, propagates subject-derived scalar maps such as AI-CBV, and generates Jacobian determinant maps for TBM.
